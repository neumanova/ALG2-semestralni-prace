# Student Budget Tracker

## Motivace
Cilem aplikace je pomoci studentum evidovat prijmy a vydaje jednoduchym zpusobem.

## Popis problemu
Program bude konzolova aplikace v jazyce Java, ktera umozni uzivateli zapisovat financni transakce, zobrazovat jejich prehled a pracovat s ulozenymi daty.

Aplikace bude umet:
- pridat prijem
- pridat vydaj
- zobrazit seznam transakci
- tridit transakce
- filtrovat transakce podle kategorie
- ukladat a nacitat data ze souboru
